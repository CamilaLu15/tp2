# Sistema de Cajero Automático - UNLaR

Simulador de un cajero automático (ATM) desarrollado en Java. El sistema procesa operaciones de manera secuencial, aplicando principios estrictos de Programación Orientada a Objetos, inmutabilidad de datos clave y un manejo robusto de excepciones personalizadas para entornos transaccionales.

## Tecnologías Utilizadas
* **Lenguaje:** Java (JDK 25)
* **Paradigma:** Programación Orientada a Objetos (POO Avanzada)
* **Tipos de Datos Críticos:** Uso de `java.math.BigDecimal` para precisión financiera
* **Entorno de Desarrollo (IDE):** Visual Studio Code

## Ejecución del Proyecto
Este proyecto está diseñado para ser ejecutado fácilmente a través de un IDE compatible con Java (como Visual Studio Code, IntelliJ IDEA o Eclipse).
